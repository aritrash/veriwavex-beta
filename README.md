# VeriWaveX - beta
**A Portable Verilog IDE for Computer Architecture Labs.**

Developed by Aritrash Sarkar. Built with Rust + egui.

## Features
- **Splash & Project Wizard**: Manage Verilog projects with `.vwx` files.
- **Integrated Simulation**: One-click compile and waveform viewing.
- **Port Wizard**: Auto-generate modules and testbenches with `$dumpfile` hooks.
